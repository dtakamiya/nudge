# Nudge UI リフレッシュ — "Hearth" デザイン

## 概要

Nudge 1on1ミーティングトラッカーの全体的なUIを刷新する。
コンセプト「Hearth（暖炉）」— 1on1の温かい対話を視覚的に表現するウォーム・オーガニックなデザイン。

## スコープ

- ライトモードのみ（ダークモード対応なし）
- 既存の全ページ・コンポーネントのビジュアル刷新
- レスポンシブ対応（モバイル・タブレット）の改善
- アニメーション・マイクロインタラクションの追加

## 3. レイアウト

### デスクトップ (>= 1024px)

```
┌──────────┬──────────────────────────────┐

```

### モバイル / タブレット (< 1024px)

```
┌────────────────────────────────────────┐
│  Top bar (ハンバーガー + ロゴ)           │
├────────────────────────────────────────┤
│  Main Content (full width, px-4)       │
└────────────────────────────────────────┘
サイドバーはオーバーレイで表示/非表示
```

### サイドバーデザイン

- 背景: `#F5EDE3`（Cream accent）
- アプリ名「Nudge」: DM Serif Display、アンバーカラー
- ナビリンク: lucide アイコン + テキスト
- アクティブ状態: アンバー背景 `rgba(217, 119, 6, 0.1)` + アンバーテキスト
- モバイル: オーバーレイ表示 + 背景暗転

### メインコンテンツ

- max-w-4xl で幅制限
- mx-auto で中央寄せ
- レスポンシブパディング: p-8 (desktop) → p-4 (mobile)

## 4. コンポーネントデザイン

### カード

- 背景: `#FFF8EF`（Surface）
- ボーダー: `#E8DDD3`
- 角丸: 16px
- シャドウ: `0 2px 8px rgba(61, 46, 31, 0.06)`
- ホバー: `translateY(-2px)` + シャドウ拡大

### メンバーカード

- イニシャルアバター: 名前ハッシュから暖色グラデーション自動生成
- メンバー名（セリフ体）、部署/役職、最終1on1日、未完了アクション数
- 「1on1を始める」アンバーボタン

### ボタン

- Primary: アンバー背景 `#D97706`、白テキスト、角丸 10px
- Secondary: 透明背景、アンバーボーダー、アンバーテキスト
- Ghost: 透明、ホバーでクリームアクセント背景
- Destructive: テラコッタ系

### バッジ

- 完了: セージグリーン背景
- 進行中: アンバー背景
- 未着手: サンド背景
- 期限超過: テラコッタ背景

### フォーム入力

- ボーダー: Sand (`#E8DDD3`)
- フォーカス: アンバーボーダー + ウォームグロー `rgba(217, 119, 6, 0.15)`
- 角丸: 10px

## 5. アニメーション

### ページ遷移

- `opacity: 0→1` + `translateY(8px→0)`
- 300ms, ease-out
- CSS アニメーションで実装

### カードインタラクション

- ホバー: `translateY(-2px)` + シャドウ拡大 (200ms)
- クリック: `scale(0.98)` (100ms)
- リスト表示: `animation-delay` によるスタガードフェードイン

### ステータス変更

- スケールバウンス `1→1.1→1` (200ms)
- 色変化にトランジション付き

### フォーム

- フォーカス: ボーダー色トランジション (200ms)
- 話題/アクション追加: スライドイン (250ms)

## 6. イニシャルアバター

メンバー名のハッシュから暖色系グラデーションを自動生成:

```typescript
const AVATAR_GRADIENTS = [
  ["#D97706", "#C27549"], // アンバー → テラコッタ
  ["#6B8F71", "#8FAF85"], // セージ → ライトセージ
  ["#B45309", "#D97706"], // ディープアンバー → アンバー
  ["#C27549", "#D4A37A"], // テラコッタ → ライトテラコッタ
  ["#7B8F6B", "#6B8F71"], // オリーブ → セージ
];
```

## 7. 技術的な注意事項

- shadcn/ui の CSS 変数をオーバーライドして実装
- 新しい shadcn/ui コンポーネントの追加は不要
- Google Fonts は `next/font/google` で最適化
- アニメーションは CSS のみ（追加ライブラリ不要）
- `tw-animate-css` は引き続き利用
